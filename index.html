About

Photo to ASCII is a free, open-source converter that turns any image into text art. It works fully in your browser using the Canvas API — no uploads, no tracking, no servers.

Built with vanilla HTML, CSS, and JavaScript. Source on GitHub.

How it works

  1. Your image is sampled at the chosen output width.
  2. Each pixel block is averaged to a brightness value.
  3. Brightness maps to a character from your chosen ramp.
  4. If color is on, each char keeps its source RGB.
